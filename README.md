# Give++
Give++ 二手交易平台

> 網站連結 : localhost:3000

- 使用者可使用功能：
  - **商品系統**：自動篩選、手動搜尋、分類搜尋，不但能瀏覽網站上所刊登的商品，且能決定搜尋結果的排序方式
  - **購物車系統**：查看欲購買的項目及內容，並且成立訂單
  - **賣家系統**：於賣家後台刊登或編輯欲銷售的商品
  - **個人資料管理系統**：編輯商店頁面的公告內容、聯絡資訊
  - **訂單系統**：隨時掌握/更新訂單的最新狀態

- 管理員可使用功能：
  - **後台管理系統**：審核最新刊登的商品，管理/維護網站的用戶及商品
